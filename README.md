## ▶️ How to Run

### Option 1: Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/shuklajaya337/MLT-Lab-Experiments.git
   cd MLT-Lab-Experiments
   ```

2. **Install required libraries**
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn jupyter
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. Open any `.ipynb` file and run the cells.

---

### Option 2: Run on Google Colab (No Installation Needed)

Click the badge below to open notebooks directly in Google Colab:
